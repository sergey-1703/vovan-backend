# Vovan Messenger

**Vovan** — это мессенджер с базовым функционалом обмена сообщениями и серверной архитектурой хранения данных.

---

## 🚀 Основные возможности

- 💬 **Мгновенные сообщения**  
  Быстрая доставка сообщений между пользователями через сервер.

- 🧑‍🤝‍🧑 **Приватные чаты**  
  Общение один-на-один между пользователями.

- 🗄 **Хранение сообщений в базе данных**  
  Все сообщения сохраняются на сервере и доступны для последующего получения.

- 🔄 **Синхронизация истории сообщений**  
  Возможность загружать историю переписки при входе с нового устройства.

- 🔐 **Аутентификация пользователей**  
  Регистрация и вход с базовой защитой аккаунта.

---

## ⚙️ Технологии

- Backend: Python (FastAPI / WebSockets)  
- Frontend: HTML, CSS, JS, Bootstrap
- База данных: PostgreSQL
