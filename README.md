# Vovan Messenger

**Vovan** — это безопасный мессенджер с упором на приватность, шифрование и полный контроль пользователя над своими данными. Проект разработан с целью обеспечить максимально защищённую коммуникацию без компромиссов.

---

## 🚀 Основные возможности

- 🔒 **End-to-End шифрование (E2E)**  
  Все сообщения шифруются на устройстве отправителя и могут быть расшифрованы только получателем. Даже сервер не имеет доступа к содержимому сообщений.

- 💬 **Мгновенные сообщения**  
  Быстрая доставка сообщений с минимальной задержкой.

- 🧑‍🤝‍🧑 **Приватные чаты и диалоги**  
  Общение один-на-один без утечек метаданных.

- 🗝 **Управление ключами на стороне клиента**  
  Ключи шифрования никогда не покидают устройство пользователя.

- 🕵️‍♂️ **Минимизация хранения данных**  
  Сервер хранит только необходимый минимум информации.

- 🔐 **Аутентификация и защита аккаунта**  
  Безопасная регистрация и вход с использованием современных криптографических методов.

---

Создать мессенджер, который:
