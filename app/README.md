# Инструкция по запуску
## С помощью Docker
1. Установить Docker
2. Запустить в терминале в корневой папке проекта
```
docker compose up --build
```
3. После запуска открыть:
-   API: http://127.0.0.1:8000
-   Swagger UI: http://127.0.0.1:8000/docs

## Вручную
1. Загрузить версию Python 3.14.3 - https://www.python.org/downloads/
2. При установке выбрать галочку "Add python to path"
3. Скачать проект
4. Настроить PostgreSQL: установить на пк postgres на сайте postgresql.org, во время установки нужно будет задать пароль. задать обязательно пароль 1234. порт обычно стоит 5432, если стоит не тот, поменять
5. После установки появится приложение pgAdmin4. войти в приложение. вставить свой код если запросят (1234)
6. Открыть servers-schemas-tables в pgAdmin4, при выполнении кода(db_main.py) в tables должны появится таблицы, если их нет, нажать правой кнопкой на tables  выбрать refresh
7. После того, как настроили PostgreSQL, нужно перейти в терминале в корневую папку проекта и выполнить 
```
py -3.14.3 -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```
6. После запуска открыть:

-   API: http://127.0.0.1:8000
-   Swagger UI: http://127.0.0.1:8000/docs
